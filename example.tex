\section{Пример ОЛМ}

Мы рссмотрим ОЛМ со следующими факторами:

\begin{itemize}
	\item Период

%АВТО
	\item Мощность авто
	\item Класс ТС
	\item Марка авто	
	\item Модель авто
	\item Возраст ТС

%СТРАХОВАТЕЛЬ
	\item Пол водителя
	\item Возраст водителя
	\item Стаж водителя


%ТЕРРИТОРИЯ
	\item Тип населённого пункта (город, деревня, село, и т.д.)
	
%ПОЛИС
	\item Страховая сумма
	
\end{itemize}
